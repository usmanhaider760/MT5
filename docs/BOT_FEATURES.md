# Bot Features

## Current Features

- Connects to MT5 through a named pipe bridge.
- Executes manual trades from the desktop UI.
- Executes trades from JSON signal payloads.
- Watches a signal folder for automated trade files.
- Validates trade direction, stop loss, take profit, lot size, and risk/reward.
- Supports spread checks, daily trade limits, and drawdown protection.
- Supports stop-loss-to-breakeven management.
- Logs trade activity and signal outcomes.
- Includes Claude-based signal analysis service.

## Planned Features

- Better setup wizard for MT5 connection.
- Safer demo/live account warnings.
- More detailed trade preview before execution.
- Strategy-level enable/disable controls.
- Backtesting and dry-run modes.
- Improved dashboard analytics.
